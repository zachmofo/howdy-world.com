# Howdy World Coding

**🤠 Practical programming for Texas ranch kids, 4-H, and future ag-tech builders.**

We're creating a free learning resource that teaches real coding skills using projects that actually matter on a farm or ranch.

## About the Project

This is the official website for **Howdy World Coding** — a movement to get rural Texas youth excited about technology through hands-on, farm-relevant programming.

### Current Status
- Static landing page (HTML + CSS)
- GitHub Pages enabled (via CNAME)

## Tech Stack
- HTML5
- CSS3 (with custom properties for easy theming)
- Fully responsive
